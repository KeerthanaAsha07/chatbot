Simple Chatbot using Python

 Project Overview
This project is a basic rule-based chatbot developed using Python.  
The chatbot interacts with users by accepting input and replying with predefined responses.  
It continues the conversation until the user enters an exit command.

---

 Features
- Greets users
- Answers simple predefined questions
- Converts user input to lowercase for better matching
- Runs continuously until user exits
- Beginner-friendly Python project



 Technologies Used
- Python

 Concepts Used
- Variables
- User Input (`input()`)
- Output Display (`print()`)
- While Loop
- Conditional Statements (`if`, `elif`, `else`)
- String Methods (`lower()`)
- Loop Control (`break`)

 How to Run This Project
